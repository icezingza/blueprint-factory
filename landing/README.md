# Landing (NamoNexus Blueprint Factory)

โฟลเดอร์นี้เป็น static website สำหรับ GitHub Pages (เส้นทาง /landing)

Structure
- index.html: Landing page + pricing cards
- topup.html: Payment page (เรียก pay(amount))
- style.css: Theme (Deep Navy + Neon Cyan + Poppins)
- js/checkout.js: เปิดลิงก์ชำระเงินและส่งกลับพารามิเตอร์ ?paid=&tx=
- logo.png: โลโก้ PNG โปร่งใส 512x512
- favicon.ico: Favicon 32x32

Deploy on GitHub Pages
1) Repo Settings -> Pages
2) Source: Deploy from a branch
3) Branch: main / folder: landing
4) Save แล้วรอสักครู่
